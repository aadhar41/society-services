# Contributing to Society ERP

Thank you for your interest in contributing to the **Society Accounting Management System (ERP)**. 

## Proprietary Software Disclaimer

Please note that this is a **Proprietary Project**.
 Contributions are strictly managed. By submitting a pull request or contributing any code, you agree to assign all rights, title, and interest in your contribution to Aadhar Gaur.

## How to Contribute

1. **Reporting Bugs**:
 Please use the standard GitHub issue tracker to report bugs. Provide as much detail as possible, including steps to reproduce.
2. **Feature Requests**: We welcome feature requests. Please detail the use case and expected benefit.
3. **Drafting Code**: 
   - Ensure the code follows PSR-12 coding standards.
   - Use consistent indentation (4 spaces).
   - Write clear, concise comments where necessary.
   - Follow the existing code structure and domain-driven design patterns.

## Security Vulnerabilities

If you discover a security vulnerability, please send an email to <aadhar.gaur@example.com> instead of opening a public issue.

Thank you!
