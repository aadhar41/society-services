Overview
--------

Society-accounting is a powerful developer toolkit for building scalable property management platforms. It combines efficient asset bundling, modular architecture, and comprehensive API support to streamline community operations. The core features include:

*   🛠️ **Asset Compilation:** Uses webpack.mix.js to optimize frontend assets for performance.
*   🚀 **Modular Architecture:** Leverages models, repositories, traits, and observers for clean, maintainable code.
*   🔐 **API & Backend:** Provides RESTful endpoints for managing societies, users, and properties with secure middleware.
*   🎯 **Automated Data Seeding:** Facilitates realistic testing with seeders for societies, blocks, plots, flats, and maintenance.
*   🧪 **Testing & Configuration:** Supports robust testing setups with phpunit.xml and environment-specific configs.
*   ⚙️ **Middleware & Security:** Implements session timeout, CSRF protection, and request handling for a secure environment.

Getting Started
---------------

### Prerequisites

This project requires the following dependencies:

*   **Programming Language:** PHP
*   **Package Manager:** Composer, Npm

### Installation

Build society-accounting from the source and install dependencies:

1.  **Clone the repository:**
    
        ❯ git clone https://github.com/aadhar41/society-accounting
        
    
2.  **Navigate to the project directory:**
    
        ❯ cd society-accounting
        
    
3.  **Install the dependencies:**
    

**Using [composer](https://www.php.net/):**

    ❯ composer install
    

**Using [npm](https://www.npmjs.com/):**

    ❯ npm install
    

### Usage

Run the project with:

**Using [composer](https://www.php.net/):**

    php {entrypoint}
    

**Using [npm](https://www.npmjs.com/):**

    npm start
    

### Testing

Society-accounting uses the {**test\_framework**} test framework. Run the test suite with:

**Using [composer](https://www.php.net/):**

    vendor/bin/phpunit
    

**Using [npm](https://www.npmjs.com/):**

    npm test
